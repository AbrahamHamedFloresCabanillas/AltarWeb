# AltarWeb - Proyecto Altares UABC

Sistema para calificar los altares de muertos de la facultad.

## Como echarlo a andar

Como la base de datos no se sube al GitHub, tienes que generarla en tu pc para que el programa no implote al iniciar.

Sigue estos pasos en cuanto clones el repo:

1. Abre el proyecto en Visual Studio.
2. Ve al menu de arriba: Herramientas > Administrador de paquetes NuGet > Consola del Administrador de paquetes.
3. En la ventana que se abre abajo, escribe este comando y dale Enter:

->   Update-Database

4. Espera a que termine (debe decir Done).
5. Dale Play (F5) al programa.
